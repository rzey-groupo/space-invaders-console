# Project Plan for Space Invaders Console

## Objective
Develop a console version of the classic Space Invaders game.

## Features
- Player can control a spaceship to shoot at incoming aliens.
- Scoring system based on how many aliens are defeated.
- Levels that increase in difficulty.
- High score tracking.

## Tasks
1. Set up project structure and initialize repository.
2. Create player controls and spaceship functionality.
3. Implement alien movement and shooting mechanics.
4. Develop scoring system.
5. Create levels and progression.
6. Test game thoroughly.
7. Prepare documentation and deployment.

## Timeline
- **Week 1**: Set up project and implement player controls.
- **Week 2**: Add alien mechanics and scoring system.
- **Week 3**: Develop levels and conduct testing.
- **Week 4**: Finalize deployment and documentation.
